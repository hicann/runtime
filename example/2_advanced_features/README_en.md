# 2_advanced_features

This chapter covers advanced Runtime capabilities, suitable for learning built-in tasks, Kernel execution, model runtime instances, callbacks, notifications, TDT, and advanced execution control after mastering basic features.

## Sample List

- [built_in_task](./built_in_task/README_en.md): Runtime built-in tasks, including Reduce and random number tasks.
- [callback](./callback/README_en.md): Stream callbacks, HostFunc callbacks, and exception callbacks.
- [ipcevent](./ipcevent/README_en.md): Cross-process Event synchronization, supporting single/multi-device multi-process wait and wake.
- [kernel](./kernel/README_en.md): Kernel binary loading, parameter organization, execution, and resource limits.
- [label](./label/README_en.md): Label, stream jumping, and device-side flow control.
- [model_ri](./model_ri/README_en.md): Model runtime instance capture, update, and switch.
- [notify](./notify/README_en.md): Inter-process Notify sharing and inter-stream synchronization.
- [tdt_channel](./tdt_channel/README_en.md): TDT Channel data transmission, capacity control, and data structure parsing.
- [tdt_queue](./tdt_queue/README_en.md): TDT Queue attribute configuration and route management.
