# tdt_channel

This directory focuses on TDT Channel basic data transmission and capacity control capabilities.

## Sample List

- [0_simple_channel](./0_simple_channel/README_en.md): Demonstrates Channel, Dataset, and DataItem construction, send, receive, and data verification.
- [1_channel_capacity](./1_channel_capacity/README_en.md): Demonstrates Channel with capacity limit, capacity query, cleanup, and graceful stop.
