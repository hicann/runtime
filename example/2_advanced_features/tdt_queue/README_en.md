# tdt_queue

This directory focuses on TDT Queue attribute configuration and route management capabilities.

## Sample List

- [0_simple_queue](./0_simple_queue/README_en.md): Demonstrates QueueAttr configuration, attribute reading, and Queue creation and destruction flow.
- [1_queue_route](./1_queue_route/README_en.md): Demonstrates queue route creation, binding, query, and unbinding.
