# 1_basic_features

本章节聚焦 Runtime 的基础编程能力，适合在完成快速入门后继续学习 Device、Context、Stream、Event 和 Memory 的核心用法。

## 样例列表

- [device](./device/README.md)：设备初始化、切换、多线程与多 Device 数据交互。
- [stream](./stream/README.md)：默认流、普通流、多流同步、失败模式控制。
- [event](./event/README.md)：事件状态、时间戳与同步行为。
- [memory](./memory/README.md)：Host/Device 数据拷贝、共享内存、IPC 与多流内存同步。

## 相关主题

- [context](./context/README.md)：Context 生命周期、切换与复用。
