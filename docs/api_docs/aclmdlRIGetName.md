## 功能说明<a name="section36583473819"></a>

获取模型运行实例的名称。如果没有调用aclmdlRISetName接口，调用本接口获取到的为空字符串。

## 函数原型<a name="section13230182415108"></a>

```
aclError aclmdlRIGetName(aclmdlRI modelRI, uint32_t maxLen, char *name)
```
