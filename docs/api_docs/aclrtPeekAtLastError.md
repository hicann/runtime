## 功能说明<a name="section93499471063"></a>

获取当前线程的Runtime（运行时管理模块）错误码，仅获取但不清空错误码。

## 函数原型<a name="section14885205814615"></a>

```
aclError aclrtPeekAtLastError(aclrtLastErrLevel level)
```
