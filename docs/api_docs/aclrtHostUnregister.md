## 功能说明<a name="section73552454261"></a>

取消注册Host内存。

本接口与[aclrtHostRegister](aclrtHostRegister.md)接口成对使用。

## 函数原型<a name="section195491952142612"></a>

```
aclError aclrtHostUnregister(void *ptr)
```
