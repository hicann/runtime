## 功能说明<a name="section36583473819"></a>

复位一个CntNotify，将CntNotify的计数值清空为0。异步接口。

## 函数原型<a name="section13230182415108"></a>

```
aclError aclrtCntNotifyReset(aclrtCntNotify cntNotify, aclrtStream stream)
```
