# 扩展接口使用说明

-   **调用接口要求**：msproftx功能相关接口须在[aclprofStart](aclprofStart.md)接口与[aclprofStop](aclprofStop.md)接口之间调用。其中配对使用的接口有：[aclprofCreateStamp](aclprofCreateStamp.md)/[aclprofDestroyStamp](aclprofDestroyStamp.md)、[aclprofPush](aclprofPush.md)/[aclprofPop](aclprofPop.md)、[aclprofRangeStart](aclprofRangeStart.md)/[aclprofRangeStop](aclprofRangeStop.md)。
-   **接口调用顺序**：**[aclprofStart](aclprofStart.md)接口**\(指定Device 0和Device 1\)--\>[aclprofCreateStamp](aclprofCreateStamp.md)接口--\>[aclprofSetStampTraceMessage](aclprofSetStampTraceMessage.md)接口--\>[aclprofMark](aclprofMark.md)接口--\>\([aclprofPush](aclprofPush.md)接口--\>[aclprofPop](aclprofPop.md)接口\)或\([aclprofRangeStart](aclprofRangeStart.md)接口--\>[aclprofRangeStop](aclprofRangeStop.md)接口\)--\>[aclprofDestroyStamp](aclprofDestroyStamp.md)接口--\>**[aclprofStop](aclprofStop.md)接口**\(与[aclprofStart](aclprofStart.md)接口的[aclprofConfig](aclprofConfig.md)数据保持一致\)。
