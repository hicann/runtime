## 功能说明<a name="section5408205113614"></a>

网络场景下，订阅算子的基本信息，包括算子名称、算子类型、算子执行耗时等。

## 函数原型<a name="section632914018717"></a>

```
aclError aclprofModelSubscribe(uint32_t modelId, const aclprofSubscribeConfig *profSubscribeConfig)
```
