# acltdtCreateQueueAttr

## 产品支持情况


| 产品 | 是否支持 |
| --- | --- |
| Atlas A3 训练系列产品/Atlas A3 推理系列产品 | √ |
| Atlas A2 训练系列产品/Atlas A2 推理系列产品 | √ |

## 功能说明

创建acltdtQueueAttr类型的数据，表示队列属性配置信息。

## 函数原型

```
acltdtQueueAttr *acltdtCreateQueueAttr()
```

## 参数说明

无

## 返回值说明

-   返回acltdtQueueAttr类型的指针，表示成功。
-   返回nullptr，表示失败。
