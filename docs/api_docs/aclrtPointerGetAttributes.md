## 功能说明<a name="section73552454261"></a>

获取内存属性信息，包括内存是位于Host还是Device、页表大小等信息。

## 函数原型<a name="section195491952142612"></a>

```
aclError aclrtPointerGetAttributes(const void *ptr, aclrtPtrAttributes *attributes)
```
