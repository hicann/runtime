## 功能说明<a name="section5408205113614"></a>

msproftx标记瞬时事件。

调用此接口后，Profiling自动在Stamp指针中加上当前时间戳，将Event type设置为Mark，表示开始一次msproftx采集。

## 函数原型<a name="section632914018717"></a>

```
aclError aclprofMark(void *stamp)
```

## 返回值说明<a name="section41841371675"></a>

返回0表示成功，返回其他值表示失败，请参见[aclError](aclError.md)。
